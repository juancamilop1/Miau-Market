Introducioon : 

Las páginas web surgieron a inicios de los años 90 con el desarrollo de la World Wide Web (WWW) ​(Wikipedia, 2021)​, creada por Tim Berners-Lee en 1989 en el CERN para facilitar el intercambio de información entre científicos ​(TELEFONIA, 2022)​. Con el tiempo, se extendieron al público, permitiendo a empresas y personas crear sitios propios. A mediados de esa década, dieron origen al comercio electrónico (e-commerce), una nueva forma de comprar y vender en línea. Aunque Internet se popularizó en los 90, la primera empresa de e-commerce, Boston Computer Exchange, fue fundada en 1982 por Alexander Randall y Cameron Hall como un mercado para vender computadoras usadas ​(Mary Gabrielyan, 2023)​. Más tarde, plataformas como Amazon y eBay impulsaron el comercio digital, transformando la forma en que los consumidores acceden a bienes y servicios y convirtiendo el e-commerce en un pilar clave de la economía global ​(Sarrah Shah, 2023)​. 

 
Resumen del poroyecto: 

El proyecto consiste en el desarrollo de una tienda virtual inteligente para mascotas que utiliza inteligencia artificial (IA) mediante la API Gemini 2.0 para ofrecer recomendaciones personalizadas de productos. A través de una breve conversación con el usuario, la IA recopila información sobre la mascota como tipo, edad, tamaño, raza y posibles alergias para sugerir los artículos más adecuados, incluyendo alimentos, juguetes, accesorios y suplementos. 

El sistema busca mejorar la experiencia de compra al brindar un servicio interactivo, rápido y adaptado a las necesidades de cada usuario y su mascota, fomentando la confianza y la satisfacción del cliente. Además, el proyecto combina tecnología de recomendación inteligente con comercio electrónico. 

Pregunta problema: 

¿Cómo diseñar e implementar una tienda virtual con un sistema de recomendación personalizado para mascotas , utilizando la API Gemini 2.0 ? 

 

Objetivo General: 

Realizar una tienda virtual inteligente para mascotas que utilice inteligencia artificial utilizando la API Gemini 2.0 para ofrecer recomendaciones personalizadas de productos según las preguntas, gustos y necesidades del usuario y según la mascota. 

La IA realizará una breve conversación con el usuario, preguntando cosas como el tipo de mascota, edad, tamaño, raza y alergias. Con base en las anteriores preguntas la IA dará los mejores productos adaptados al cliente (como alimento, juguetes, accesorios y suplementos). 

  

Objetivos Especificos: 

Diseñar la interfaz y estructura de la tienda virtual para ofrecer una navegación sencilla, agradable y atractiva al usuario. 

Implementar la API Gemini 2.0 para integrar un sistema de inteligencia artificial capaz de interactuar con el usuario mediante preguntas. 

Desarrollar un módulo de recomendaciones que analice las respuestas del usuario y sugiera productos personalizados según su tipo de mascota. 

Clasificar los productos por categorías (alimentos, juguetes, accesorios y suplementos) para mejorar la experiencia de búsqueda. 

Evaluar la precisión y eficacia del sistema de recomendaciones mediante pruebas con distintas mascotas. 

Tecnologias que se van a usar: 

Para el desarrollo del proyecto se emplearán diversas herramientas y tecnologías que permitirán garantizar un funcionamiento eficiente y seguro: 

Angular: Se utilizará como framework frontend para construir una interfaz moderna, dinámica e intuitiva, ofreciendo una experiencia de usuario fluida. 

Django: Actuará como framework backend para gestionar la lógica del servidor, las rutas, la autenticación y la comunicación con la base de datos. 

MySQL Server: Será el sistema de gestión de bases de datos encargado de almacenar la información de usuarios, productos, mascotas y recomendaciones generadas. 

API Gemini 2.0 (IA): Se integrará para dotar al sistema de inteligencia artificial conversacional, permitiendo realizar preguntas al usuario y ofrecer recomendaciones personalizadas basadas en sus respuestas.
